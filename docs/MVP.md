# SmartExpense Pro - MVP

## Goal
Build a privacy-first expense and subscription manager for Indian students and young professionals, with basic AI-assisted categorization.

## MVP Features
- User signup & login (JWT-based auth)
- Manual expense entry (amount, category, description, date, payment method)
- View list of expenses with total for a selected date range
- Predefined categories (Food, Travel, Rent, Subscriptions, Shopping, Others)
- Manual subscription tracking (name, amount, billing cycle, next renewal date)
- Simple summary: total expenses this month + total subscriptions
- Basic AI endpoint to guess category from description text

## Non-Goals (for MVP)
- No bank login or card details
- No automatic bank integration
- No mobile app yet
- No advanced AI analytics yet
